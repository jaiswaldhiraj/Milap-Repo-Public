
# Milap-Public-Repo
Public showcase of Milap, a dating app built with Android Studio using Java, Firebase Realtime Database, and FCM. Features include user onboarding, profile setup, swipe cards, mutual likes, notifications, and chat.

## üöÄ Features
- **Phone Number Authentication** ‚Äì Secure login via OTP verification to ensure authenticity.  

- **Personalized Profiles** ‚Äì Users create detailed profiles with personal information, preferences, and multiple photos.  

- **CardStack Swipe Interface** ‚Äì Swipe right to like, left to pass, with smooth animations powered by CardStackView and Lottie. 

- **Smart Profile Display** ‚Äì Each card highlights a profile photo, age, bio, and randomly selected hobbies for freshness.  

- **Photo Upload & Cropping** ‚Äì Upload and crop up to 4 images during onboarding.

- **Real-Time Database Integration** ‚Äì Powered by Firebase Realtime Database for instant syncing of user profiles, likes, and matches.  

- **Like & Match System** ‚Äì Swipe right to like, and when two users like each other, they form a match.  

- **Notifications & Messaging** ‚Äì Users get notified when someone likes them, with a built-in chat system for matched users.  

- **Dynamic User Discovery** ‚Äì Fetches all user profiles at startup, displaying them in a stack for quick browsing.

## ‚öôÔ∏è Tech Stack

- **Frontend (Android)** ‚Äì Java, XML, Lottie Animations   

- **Backend** ‚Äì Firebase Realtime Database & Firebase Authentication  
 
- **Media Handling** ‚Äì Glide for image loading, built-in cropping tools for photo uploads  

- **Cloud Functions** ‚Äì Used for match-making automation (triggered when two users like each other)  

## üöÄ Future Scope
- AI-powered match recommendations based on user preferences & hobbies  

- Advanced filters (location, interests, lifestyle choices)  

- Push notifications for messages and new matches  

- Profile verification badges
