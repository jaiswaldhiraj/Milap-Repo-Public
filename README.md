# 💖 Milap – Public Repo  

✨ _Find Your Match with **Milap** – A modern dating app for meaningful connections._ ✨  

🚀 **Milap** is a modern dating app built with **Android Studio (Java)** and powered by **Firebase Realtime Database & FCM**.  
It offers smooth onboarding, swipe cards, real-time chat, and intelligent matchmaking – designed to bring people closer in the digital world.  

---

## ✨ Features  

- 📱 **Phone Number Authentication** – Secure login via OTP for verified access.  
- 🧑‍🤝‍🧑 **Personalized Profiles** – Add details, preferences, bio, and up to 4 profile photos.  
- 🎴 **Swipe Cards (Tinder-style)** – Smooth CardStackView + Lottie animations for like/pass gestures.  
- 🎲 **Smart Profile Display** – Fresh cards with random hobbies shown dynamically.  
- 📸 **Photo Upload & Cropping** – Built-in image cropping and multiple uploads.  
- ⚡ **Real-Time Sync** – Firebase keeps profiles, likes, and matches updated instantly.  
- 💌 **Like & Match System** – Mutual likes automatically become matches.  
- 🔔 **Notifications & Messaging** – Instant alerts + chat system for matched users.  
- 🌍 **Dynamic Discovery** – Browse all user profiles at startup with real-time updates.  

---

## 🛠 Tech Stack  

- 🎨 **Frontend (Android)** – Java, XML, Lottie Animations.  
- 🔥 **Backend** – Firebase Realtime Database, Authentication, Firebase Functions using node.js, Firbase Cloud Messaging & Firestore.
- 🖼 **Media Handling** – Glide + cropping tools for photo uploads.
- ⚙️ **Cloud Functions** – Automates matchmaking on mutual likes.
